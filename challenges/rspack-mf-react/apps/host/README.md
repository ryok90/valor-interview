# Rspack project

## Get started

Start the dev server, and the app will be available at [http://localhost:4200](http://localhost:4200).

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
